Грамматика языка основана на грамматики pl0.
На данном этапе разработке:
распознавание токенов не зависит от регистра;
поддерживаемые типы данных: STRING, INTEGER, FLOAT, BOOL;
переменные объявляются в начале процедуры или программы;
символ присвоения "=";
операторные скобки begin end;
каждое действие отделяется ";";
конец программы обозначается ".";
поддерживаются коментарии различной длинны (однострочные и многострочные);
поддерживаются условный оператор if и оператор цикла while
	if условие then действие
	while условие do действие
логические и арифметические операции: ==, >, <, <=, >=, !=, +, -, *, /, and, or;
каждое отдельное условие заключается в скобки;
